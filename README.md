# Rehabilus Fleet Quota Management System

## Project Overview
This project is designed to manage the quota systems for the Rehabilus fleet. It enables tracking and management of fleet quotas, ensuring effective utilization and management of resources.

## Features
- User Management
- Quota Tracking
- Reporting and Analytics
- Notifications and Alerts

## Installation
### Prerequisites
- Node.js
- MongoDB or SQL Database

### Steps
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Configure the database settings in the `.env` file.
4. Run the application using `npm start`.
